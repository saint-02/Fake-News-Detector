# Fake News Detector (Prototype)

## Description
A beginner-friendly web application prototype that analyzes news text using simple keyword-based logic to estimate whether news content may be fake or real.

## Features
- Paste news text
- Keyword-based analysis
- Displays credibility result

## Tech Stack
- HTML
- CSS
- JavaScript

## Status
Prototype / Starter version

## Disclaimer
This project provides an estimation only and should not replace professional fact-checking.
